## Установка 

1. Клонирование репозитория 

```git clone https://github.com/Unicorn-Naughty/ind-test.git```

2. Переход в директорию torgbox-test

```cd torgbox-test```

3. Установка зависимостей

```npm i ```

4. Запуск для демонстрации возможностей torgbox-test

```npm start```
